## 上机考试管理系统

#### 项目描述

该项目为大三上学期（2019-09）创新实践课程大作业，是一个面向本人所在大学全体师生的考试管理系统，教师负责发布考试以及后续相关管理，学生负责登录系统参与相应的考试，部分教师账号拥有管理员权限，可管理学生、教师、考试等相关信息。

#### 所用技术

* 数据库使用 Mysql 8.0
* 基于 Bootstrap4 编写前端页面
* 基于 SSM 框架实现后端代码

#### 实现功能

* 教师端：登录、创建考试、Excel 批量导入考生名单、管理考试、实时通知、试卷压缩归档等。
* 学生端：登录（密码由 MD5 加密）、下载试卷、接收通知（Ajax 轮询）、上传试卷、修改密码等。 
* 管理员：：学生管理、教师管理、考试管理等。
